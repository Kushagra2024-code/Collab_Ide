---

### Prompt

I want to build a **real-time collaborative coding platform** similar to VS Code Live Share or Replit Multiplayer, where multiple developers can work on the same project simultaneously.

## Goal

Build a full-stack web application that allows teams to collaborate on coding projects in real time while tracking every user's activity.

### Tech Stack

Use:

* Frontend: React + TypeScript + Tailwind CSS
* Backend: Spring Boot (Java)
* Database: PostgreSQL
* Authentication: JWT + OAuth (Google/GitHub)
* Real-time Communication: WebSockets (Socket.IO or Spring WebSocket)
* Code Editor: Monaco Editor (VS Code editor)
* Redis (optional for scaling)
* Docker + Docker Compose
* Nginx for deployment

---
